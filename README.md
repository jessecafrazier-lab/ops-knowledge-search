# Ops Knowledge Search
**Client Operations × Account Management — Internal AI Tool**

A self-service search tool that lets the Account Management team find Client Ops processes, guides, and protocols instantly — without needing to ping anyone directly.

Built by Jesseca (Operations Specialist) as part of **Pillar 03: Shared Knowledge & Documentation** within the AM OKR Collaboration Playbook.

---

## What it does

Ask a question in plain English and get an answer grounded only in official Ops resources. It does not use general AI knowledge — if the answer isn't in the loaded documents, it says so and directs you to the right person.

**If a question can't be answered from the resources, it routes you to:**
- Beth — Velocity
- Emily — Strategy
- Nichole — Production
- Josue — Automation

---

## How to use it

1. Open `Ops_Knowledge_Search.html` in any browser
2. Type your question in the search bar (or tap a quick-question chip)
3. The AI answers using only the resources in the library below the search bar

---

## How to add resources (Ops team only)

Three ways to add documents to the knowledge base:

| Method | When to use |
|---|---|
| **URL** | Paste a Confluence, Jira, Salesforce, or Google Docs link |
| **Paste text** | Copy-paste content from a doc that isn't published yet |
| **Upload file** | Upload a TXT, DOCX, or MD file directly |

> **Note:** Resources are session-based — they reset when the browser is closed. For a persistent version with saved resources, contact Jesseca.
